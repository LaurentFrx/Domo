# Domo

Tableau de bord énergie de la maison Feroux.
PWA SvelteKit 2 déployée sur https://domo.feroux.fr

## Stack

SvelteKit 2 · Svelte 5 Runes · TypeScript · Tailwind 4 · LayerCake · @vite-pwa/sveltekit · Docker · Caddy

## Dev local

```bash
pnpm install && pnpm dev
```
